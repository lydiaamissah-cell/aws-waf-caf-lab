# AWS Well-Architected & Cloud Adoption Framework Lab

## Overview
This project evaluates a two-tier web application migration from on-premises infrastructure to AWS using the AWS Well-Architected Framework (WAF) and the AWS Cloud Adoption Framework (CAF).

## Approach

### Task 1 – Architecture Review
Identified workload components and architectural weaknesses including single points of failure, lack of backups, limited scalability, and security gaps.

### Task 2 – WAF Assessment
Evaluated the workload across:
- Operational Excellence
- Security
- Reliability
- Performance Efficiency
- Cost Optimization

Recommended AWS services to address identified weaknesses.

### Task 3 – CAF Assessment
Analyzed organizational readiness using:
- Business
- People
- Governance
- Platform
- Security
- Operations

Identified key enablers required for successful migration.

### Task 4 – Improved Architecture
Designed a multi-AZ, highly available architecture including:
- VPC with public/private subnets
- Application Load Balancer
- EC2 Auto Scaling
- Amazon RDS (Multi-AZ)
- Monitoring and security controls
